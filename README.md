# gulp-base

## Get Started
```
npm install
```

## Usage
```
npm start
```
- Start BrowserSync
- Start Watch files
- Compile Sass
- Compile Jade
- spritesmith
- image minify
- Create icon font

### Use individually
__Compile Jade__
```
gulp html:pre
```

__Compile Sass__
```
gulp css:pre
```

__spritesmith && image minify__
```
gulp image
```

__Create icon font__
```
gulp font
```

__Create JavaScript Document__
```
gulp js:doc
```

__Create Difference file__
```
gulp diffzip
```
